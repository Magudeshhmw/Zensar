Section 01
What is SkillMatrix Pro?
A web application that replaces the current Excel skill tracking sheet. Every QA team member fills their skill ratings online. Managers see a real-time dashboard with charts, gaps, and team performance — without asking anyone for an update.
🎯 Core Problem Solved

Currently the team uses an Excel file. It gets outdated, is hard to filter, and managers cannot see team-wide gaps instantly. This app fixes all of that with a live, filterable, visual system that works on mobile and desktop.

Before — Excel sheet
Manual updates, gets outdated, no charts, no filters, shared via email, breaks on mobile, no gap detection.
After — SkillMatrix Pro
Real-time updates, auto charts, smart filters, gap analysis, growth tracking, works on all devices.
Extra value added
AI course recommendations, manager-employee chat, notification system, downloadable reports.

🔐 How Access is Controlled

In Supabase, every user has a role column — value is either employee or admin. Supabase Row Level Security (RLS) rules make it so employees can only read their own data rows. Admin pages are not rendered in the browser for employee accounts — they are completely hidden, not just locked.

Section 07
Chat System in Admin Panel(Optional)
Managers can message any employee about skill ratings, growth plans, or training. Employees reply from their portal. All conversations are visible and filterable in the admin panel.
💬 How Chat Works

The admin panel has a Chat section. It shows a list of all employees on the left. Click any employee to open the conversation thread on the right. Manager types a message → saved to Supabase messages table → employee sees it in their portal with a notification badge. Employee replies → manager sees the reply in admin panel.

📋 What Chat is Used For

Rating queries: "Your Playwright rating is Level 1 — do you need training resources?"
Growth plan follow-up: "Your Selenium plan is overdue — please update your status."
Announcements: "Please complete your Q2 skill update by March 31."
Approval: "Your skill ratings have been reviewed and approved by manager."

✅ Chat Features in Admin Panel

• Filter chats by: Unread, Department, Date range, Employee name
• Mark individual chats as Read / Unread
• See online status of employee (last seen)
• Search inside any conversation
• Bulk message: send same message to all employees or a department
• All messages are timestamped and stored permanently in Supabase

Section 13
Supabase Sleep Problem — Fix
⚠️ What is the sleep problem?

Supabase free tier pauses your database if no one uses the app for 7 days. When it wakes up the first request takes 10–30 seconds. Users see a blank screen. This looks like the app is broken but it is not — the database is just sleeping.

✅ Solution 1 — Keep-alive ping (Free)
Create a GitHub Actions workflow that runs every 3 days and sends a simple SELECT query to Supabase. This costs zero money and keeps the database awake permanently. Takes 10 minutes to set up.
💳 Solution 2 — Supabase Pro ($25/month)
Upgrading to Supabase Pro removes the sleep completely. Database is always on. Recommend this once the team is actively using the app daily.
⚡ Solution 3 — Vercel cron (Free)
If deploying on Vercel, add a Vercel Cron function that pings the Supabase REST endpoint every 2 days automatically. No GitHub needed.
✅ Recommended for this project: Solution 1 + Solution 2 when approved

Use the free GitHub Actions ping during development. Once manager approves and team starts using daily, upgrade to Supabase Pro. Cost is ₹2,100/month approximately.
